### Setup terminal Promt

#### Install Powerline


Add to your `.bashrc`

```bash
# install powerline 
if [ -f ~/.powerline-bash.sh ]; then
    . ~/.powerline-bash.sh
fi
```

More info -> https://github.com/JonnyFox/powerline-bash
